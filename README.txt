CellTracks5_1.py is a class for the cell "forest" object to represent a set of tracks as a graph and generate visuals (e.g. networkx visualization, images with bounding boxes)

run_viterbi_9_1.py loads data, runs the viterbi algorithm implementation using functions in the CellTracks5_1.py class and generates visualizations

utility_functions_5.py provides functions for loading data, creating binary segmentation masks and generating contour feature vectors
